Instalando Mongo DB en Fedora

Esta entrada es un pequeño conjunto de tips para instalar y acceder a Mongo DB en Fedora 20.

Instalación:

sudo yum install -y mongodb-org

Ejecución:

sudo systemctl start mongod

Chequear estatus:

sudo systemctl status mongod

Consola de Mongo DB:

mongo

Detención:

sudo systemctl stop mongod
