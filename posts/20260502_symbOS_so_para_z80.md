# SymbOS, ¡un sistema operativo con interface gráfica y multitasking para el Z80!

Leyendo las novedades del proyecto [ZX Spectrum Next](https://www.kickstarter.com/projects/spectrumnext/zx-spectrum-next-issue-3-0/posts/4676856), puntualmente la actualización 18, acabo de descubrir que existe un sistema operativo gráfico multitarea para el [Z80](https://es.wikipedia.org/wiki/Zilog_Z80), [SymbOS](https://www.symbos.org/). Programado en ensamblador, está compilado para múltiples plataformas, pudiendo ejecutarse las aplicaciones en varios microcomputadores de los 80s. La Wikipedia tiene una [buena entrada](https://es.wikipedia.org/wiki/SymbOS), así como [Hackaday](https://hackaday.com/2025/07/19/symbos-is-a-funhouse-mirror-look-at-a-future-that-never-was/) y [El laboratorio de Oscar](https://sites.google.com/site/ellaboratoriodeoscar/8-bits/--symbos). Mas interesantemente, al menos para mi, según la mencionada actualización de _ZX Spectrum Next_, ¡hay un port para la [ZX Spectrum](https://es.wikipedia.org/wiki/Sinclair_ZX_Spectrum)!
