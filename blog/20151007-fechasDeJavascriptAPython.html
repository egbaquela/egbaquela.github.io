Importando fechas desde Javascript a Python

Estuve trabajando en una aplicación en Python que toma como dato de entrada un JSON con fechas, y necesito transformarlo en el formato de objetos 'datatime' de Python.

Basicamente tengo esto:

'Wed Aug 19 08:10:00 GMT-0300 2015'

Y quiero convertirla en esto:

datetime.datetime(2015, 8, 19, 8, 10, tzinfo=datetime.timezone(datetime.timedelta(-1, 75600), 'GMT'))

El código necesario es:

import datetime
datetime.datetime.strptime('Wed Aug 19 08:10:00 GMT-0300 2015', '%a %b %d %H:%M:%S %Z%z %Y')
