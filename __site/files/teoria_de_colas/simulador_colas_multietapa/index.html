Simulador de colas multietapa — M/M/c y G/G/c en serie

Construí una línea de atención o producción con cualquier cantidad de etapas, definí servidores paralelos por etapa, tasa de arribo, distribuciones de servicio y múltiples semillas. El simulador ejecuta eventos discretos, compara corridas, muestra la evolución temporal de colas y WIP, y permite reproducir la trayectoria evento por evento.

Constructor visual

Arrastrá una etapa al lienzo, reordená las estaciones y editá los parámetros en la tabla.

⚙️ Etapa / Delay
Constructor simplificado: el flujo es serial y se conectan automáticamente Source → Queue/Server₁ → … → Queue/Serverₙ → Sink. Cada etapa representa una cola FIFO más c servidores paralelos.

Parámetros generales

Podés correr una sola trayectoria o varias semillas para comparar variabilidad.

Corridas independientes.
Permite reproducir una corrida específica.
Afecta sólo la animación.
Para tener en cuenta: cuando todas las distribuciones son exponenciales, la salida de una etapa M/M/c estable puede tratarse como Poisson, lo que conecta con Burke. Cuando pasamos a G/G/c, la simulación se vuelve el objeto principal de análisis.
