# Lecturas sobre Dinámica de Sistemas - Marzo 2026

* [A Categorical Framework for Modeling with Stock and Flow Diagrams](https://arxiv.org/abs/2211.01290): un acercamiento a dinámica de sistemas desde la teoría de categorías. Bastante prometedor, y con [paquete en Julia](https://github.com/AlgebraicJulia/StockFlow.jl).
* [Towards explainable decision support using hybrid neural models for logistic terminal automation](https://arxiv.org/abs/2509.07577v2): sobre la integración de dinámica de sistemas y deep learning para problemas de logística.
* [Compositional System Dynamics: The Higher Mathematics Underlying System Dynamics Diagrams & Practice](https://arxiv.org/abs/2509.18475v1)
* [Extending System Dynamics approach to higher index DAE](https://proceedings.systemdynamics.org/2011/proceed/papers/P1147.pdf)
* [Especificación de XMILE](https://docs.oasis-open.org/xmile/xmile/v1.0/os/xmile-v1.0-os.html)
* [Leveraging Loop Polarity to Reduce Underspecification in Deep Learning](https://onlinelibrary.wiley.com/doi/10.1002/sdr.70014)
* [Redes neuronales para predecir sistemas dinámicos](https://informatica.vu.lt/journal/INFORMATICA/article/1388/text): no es SD, pero relacionado.
* [Deep backward stochastic differential equation method](https://en.wikipedia.org/wiki/Deep_backward_stochastic_differential_equation_method): entrada de wikipedia útil para explicar desde cero como funciona el BSDE.
