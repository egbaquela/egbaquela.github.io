# ¿Como convertir un archivo markdown a PDF en _Fedora Linux_?

Vamos a utilizar [_pandoc_](https://pandoc.org/). Para ello, en la terminal, lo instalamos con el siguiente comando:

```
sudo dnf install pandoc texlive pdflatex
```

Y luego hacemos:

```
pandoc input.md -o output.pdf
```

(y con el mismo _pandoc_ podemos convertir a html, por ejemplo).
