Links sobre el lenguaje Julia

El objetivo de este post es ir recolectando links útiles para el auto-aprendizaje de Julia. No solo para aprender cosas que todavía no se, sinó también para mis clases, siempre hay material interesante que aporta una nueva visión sobre algo que uno ya conoce. A medida que encuentre nuevo material lo voy a ir subiendo.

  1. Ciencia de datos con Julia
  2. Buenas prácticas
  3. Performance
  4. Crear publicaciones (libros, webs) con Julia
  5. Interface Julia-Python
  6. Static compilation

  • Repo del libro Julia Data Science (en inglés, repositorio): https://github.com/JuliaDataScience/JuliaDataScience

  • Blog (desactualizado) sobre el uso de Julia para ciencia de datos (en inglés, blog): https://www.juliafordatascience.com/
