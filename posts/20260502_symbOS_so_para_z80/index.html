SymbOS, ¡un sistema operativo con interface gráfica y multitasking para el Z80!

Leyendo las novedades del proyecto ZX Spectrum Next, puntualmente la actualización 18, acabo de descubrir que existe un sistema operativo gráfico multitarea para el Z80, SymbOS. Programado en ensamblador, está compilado para múltiples plataformas, pudiendo ejecutarse las aplicaciones en varios microcomputadores de los 80s. La Wikipedia tiene una buena entrada, así como Hackaday y El laboratorio de Oscar. Mas interesantemente, al menos para mi, según la mencionada actualización de ZX Spectrum Next, ¡hay un port para la ZX Spectrum!
