Links acerca de licencias para libros y blogs

Dejo acá unos links, porque se que en algún momento los voy a necesitar, sobre licencias del estilo open-access para libros, blogs, etc:

  • Choose an open source license: guía para elegir licencias. Útil para software también.

  • Creative Commons Licenses for GitHub Projects: repositorio de Github con texto y código markdown para incluir licencias de Creative Commons en proyectos de Github.

  • Choosing a license for an open-source book: dicusión acerca de licencias para libros de acceso abierto.

  • License Chooser: aplicación para seleccionar la licencia Creative Commons mas apropiada.
