¿Como convertir un archivo markdown a PDF en Fedora Linux?

Vamos a utilizar pandoc. Para ello, en la terminal, lo instalamos con el siguiente comando:

sudo dnf install pandoc texlive pdflatex

Y luego hacemos:

pandoc input.md -o output.pdf

(y con el mismo pandoc podemos convertir a html, por ejemplo).
