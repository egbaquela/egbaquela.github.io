Simulador DBR (Drum‑Buffer‑Rope) — línea de 5 servidores en serie

Simulá una línea con arribos Poisson y tiempos entre servicios exponenciales. Podés elegir cuál estación será el Drum, fijar el Buffer protegido y observar cómo la Rope libera trabajo para sostener ese buffer. El simulador también muestra qué pasa cuando el drum elegido no coincide con la verdadera restricción.

Parámetros del sistema

Se modela demanda externa con tasa λ. La cuerda libera trabajo al sistema para sostener el buffer del drum considerando el WIP ya liberado aguas arriba.

Tasas de servicio μᵢ
Mecánica DBR implementada
• La demanda llega a un backlog externo.
• La cuerda libera trabajo para sostener el buffer del drum en base al WIP ya liberado aguas arriba.
• Todas las estaciones tienen colas antes del servidor, incluso si el drum fue mal elegido.
• Si el drum o el buffer están mal, el buffer visible frente al drum puede vaciarse, pero nunca se vuelve negativo.
